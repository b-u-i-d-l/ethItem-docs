Description

All the wrapped ERC721 NFTs will be created following this Model.

The minting operation can be done only by transferring the original ERC721 Item through the classic safeTransferFrom call.

The burning operation will send back the original wrapped NFT.

To initialize it, the original 'init(address,string,string)' function of the EthItem Token Standard will be used, but the first address parameter will be the original ERC721 Source Contract to Wrap, and NOT the ERC20Model, which is always taken by the Contract who creates the Wrapper.
